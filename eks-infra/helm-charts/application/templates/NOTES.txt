1. Get the application pods:

  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "application.name" . }}

2. Get the service:

  kubectl get svc -n {{ .Release.Namespace }} {{ include "application.fullname" . }}

3. If using Ingress + AWS Load Balancer Controller, find the ALB hostname:

  kubectl get ingress -n {{ .Release.Namespace }} {{ include "application.fullname" . }}

Then hit:

  http://<ALB-HOSTNAME>/
